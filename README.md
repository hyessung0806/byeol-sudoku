# 별 사냥 스도쿠 — 설치 방법

이 폴더의 파일들은 모두 **같은 폴더 안에** 함께 있어야 합니다.
- index.html  (앱 본체)
- manifest.json  (앱 정보)
- sw.js  (오프라인 실행용)
- icon-192.png / icon-512.png / icon-maskable-512.png  (앱 아이콘)

## A. 가장 쉬운 방법 — GitHub Pages (무료, HTTPS)

1. github.com 에서 새 저장소(repository)를 만듭니다. 예: `byeol-sudoku` (Public)
2. 이 폴더의 6개 파일을 저장소에 올립니다. (웹에서 "Add file → Upload files"로 끌어다 놓아도 됩니다)
3. 저장소 → Settings → Pages → "Build and deployment"의 Source를 **Deploy from a branch**,
   Branch는 **main / (root)** 로 두고 Save.
4. 1~2분 뒤 `https://<내아이디>.github.io/byeol-sudoku/` 주소가 생깁니다.

## B. 갤럭시탭에 설치하기

1. 갤럭시탭 **크롬**에서 위 주소를 엽니다.
2. 우측 상단 메뉴(⋮) → **"앱 설치"** 또는 **"홈 화면에 추가"** 를 누릅니다.
3. 홈 화면에 별 아이콘이 생기고, 누르면 주소창 없이 전체화면으로 실행됩니다.
4. 가로로 돌려서 사용하세요. (가로 화면에서 왼쪽 사이드바 + 오른쪽 보드로 보입니다)

## 사용 방법

- **단계**: 6×6부터 시작합니다. 한 난이도에서 50문제를 풀면 다음 난이도가, 한 단계를 모두 깨면 다음 크기(9×9 → 12×12)가 열립니다.
- **오늘의 미션**: 하루 2문제. 2문제를 풀면 그날 달력에 도장(★)이 찍히고 연속 일수가 올라갑니다. 더 풀어도 됩니다.
- **힌트**: 6×6은 1번, 9×9는 2번. 정답을 알려주지 않고 "지금 가장 쉬운 칸"을 짚어 풀 길만 안내합니다.
- **미션 기록 탭**: 도장 달력과 연속 기록, 총점을 봅니다.

## 설정 바꾸기 (index.html 위쪽 상수)

- LEVEL_TARGET : 한 난이도 통과에 필요한 문제 수 (현재 50)
- DAILY_TARGET : 하루 미션 문제 수 (현재 2)
- POINTS       : 한 문제 점수 (현재 20000)
- STAGES 배열에 한 줄 추가하면 새 단계가 생깁니다. 예) { size:16, label:"16×16", boxRows:4, boxCols:4 }

※ 파일을 고쳐 다시 올린 뒤 변경이 바로 안 보이면, sw.js의 `byeolsudoku-v1`을 `v2`로 바꿔 올리고 앱을 한 번 닫았다 켜세요.
